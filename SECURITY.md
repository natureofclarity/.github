# Security Policy

## Reporting a vulnerability

For security issues affecting any repository in this organization, email
[michael@natureofclarity.com](mailto:michael@natureofclarity.com) directly.
Do NOT open a public issue.

Acknowledgement target: within 72 hours.

## Supported versions

Most repos in this org are personal infrastructure or active development.
Security patches land on `main` of the affected repo; no formal backport policy.
