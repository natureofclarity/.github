## Conventions

- Trunk-based development for personal infrastructure repos.
- main / staging / dev branches for deployed services with GitHub Environments for per-env secrets.
- Per-repo pre-commit checks where applicable (sync, ledger verify, tests, lint).
- Structural decisions logged as DEC-NNN entries in each repo's _system/decisions/.
